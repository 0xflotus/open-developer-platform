title: FINOS Maintainers Cheatsheet
---

The following table guides FINOS Maintainers on key responsibilities and where to find helpful resources. For everything else, refer to [odp.finos.org](https://odp.finos.org/) or [open a issue on ODP](https://github.com/finos/open-developer-platform/issues/new).

| Maintainer Responsibility | How FINOS supports you |
|:---|:---|
| Integrate Continuous Integration and Deployment into FINOS Project Builds. | [Use the FINOS Open Developer Platform for Continuous Integration and Deployment.](https://odp.finos.org/docs/development-infrastructure/continuous-integration/intro/) |
| Publish Documentation from GitHub Repositories to FINOS Project Websites. | [Use Docusauraus and GitHub Actions for Publishing Project Documentation.](https://odp.finos.org/docs/development-infrastructure/project-documentation#docusaurus) |
| Run Project Meetings, Including Creating Agendas, Recording Attendance and Publishing Minutes. | <p>[Follow FINOS Meeting Procedures from the FINOS Community Governance Repository.](https://github.com/finos/community/blob/master/governance/Meeting-Procedures.md)</p><p>[Run Project Meetings from GitHub Issues using the Meeting Attendee Tracker Action.](https://odp.finos.org/docs/project-collaboration#meeting-minutes)</p> |
| Prevent, discover and responsibly disclose Security Vulnerability Issues. | <p>[Use the FINOS Open Developer Platform for Project Security Vulnerability Scanning.](https://odp.finos.org/docs/development-infrastructure/code-validation/whitesource)</p><p>[Follow the FINOS responsible disclosure policy.](https://finosfoundation.atlassian.net/l/c/gTEPn6J1)</p> |
| Grow maturity, viability and value of your project. | [Learn about the project lifecycle and drive your project toward activation.](https://github.com/finos/community/blob/master/governance/Software-Projects/Project-Lifecycle.md) |
